Please ignore the NUCLEO-G474RE and llvm-project folder. 

## Usage
1. Download STM32 CUBE IDE
2. Open the PWR_CurrentConsumption with STM32 CUBE IDE
3. Follow the standard instruction of STM32 CUBE IDE
4. Acquire NUCLEO-G474RE and STLINK-V3PWR and follow the documentation for hardware connection and setup.
